# Early Detection Of Depression From Social Media Posts

A Machine Learning and Django-based web application that predicts signs of depression from social media posts using Natural Language Processing (NLP) techniques.

## Project Overview

This project analyzes user text data from social media posts and predicts whether the content indicates depression. The system uses Machine Learning algorithms and NLP preprocessing techniques to classify the text.

The application is developed using:

- Python
- Django
- Machine Learning
- NLP (Natural Language Processing)
- HTML, CSS, Bootstrap
- SQLite/MySQL

---

## Features

- User Registration and Login
- Admin Dashboard
- Depression Prediction from Text
- NLP Text Preprocessing
- Machine Learning Classification
